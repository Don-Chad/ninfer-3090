# Pre-Optimization Documentation Archive

This directory contains completed planning, design, and stage-gate material from before the
optimization phase.

Archived documents are historical records. They describe how earlier milestones were planned and
verified, but they are not current work entrypoints unless a current document explicitly cites them.

  starting baseline.

  reference.

  reference.
